# PequeMente

Prototipo de aplicación educativa para niños desde los 3 años.

## Objetivo
Desarrollar habilidades cognitivas, motoras y de resolución de problemas mediante juegos simples y progresivos.

## Incluye
- Juego de conteo con retroalimentación inmediata.
- Juego de reconocimiento de colores.
- Propuesta de progresión por niveles de edad.

## Ejecutar
Abre `index.html` directamente en el navegador o usa un servidor local:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
